# Landing Propuesta Comercial - Diana Boldizar

Landing one-shot en React + Vite, mobile-first, con hero en video y estructura por bloques.

## Requisitos

- Node.js 18+
- npm

## Desarrollo local

```bash
npm install
npm run dev
```

## Build de producción

```bash
npm run build
```

El resultado queda en `dist/`.

## Despliegue en Hostinger con GitHub

1. Sube este proyecto a un repositorio en GitHub.
2. En Hostinger, crea un sitio tipo **Static front-end application**.
3. Conecta el repositorio y selecciona la rama principal.
4. Build command: `npm install && npm run build`
5. Publish directory: `dist`
6. Activa despliegue automático para que cada push publique cambios.
